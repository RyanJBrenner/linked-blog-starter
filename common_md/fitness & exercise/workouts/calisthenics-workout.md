# Calisthenics Workout (bar workout)

**3 Rounds of:**

10-8-6-4-2
	Pull up
	Burpee
	Knee Raise
	Static Lunge
    
For a total of 400 reps
	100 pull ups
	100 burpees
	100 knee raises
	100 static lunges
